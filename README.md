# JargonLang
Version 0.7

to use, write your code as a command line argument.\

Values: int, float
# Operations: 
## Addition `a+b`
Adds `a` and `b`
## Subtraction `a-b`
Subtracts `b` from `a`
## Negation `-a`
Returns the additive inverse of `a`
